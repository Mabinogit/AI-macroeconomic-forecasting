# Economic Indicator Prediction

## Overview
This project explores the use of machine learning and time-series methods
to predict key economic indicators such as GDP growth, inflation, and
unemployment using historical macroeconomic data.

## Motivation
Accurate forecasting of economic indicators is essential for policy analysis,
business planning, and investment decision-making. This project focuses on
building interpretable and compute-efficient models suitable for real-world
economic data.

## Scope
- GDP growth
- Inflation
- Unemployment rate
